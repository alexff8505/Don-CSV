# Repository Guidance

## macOS UI

- Use macOS conventions wherever possible.
- Use basic, stock macOS Tahoe 26 and later GUI elements and platform-standard processes wherever possible.
- Prefer standard SwiftUI and AppKit controls, layouts, behaviours, and system styling.
- Refrain from custom-drawn or custom-styled UI when a stock macOS equivalent exists.
- Keep tabs, sidebars, toolbars, menus, dialogs, and interactions native and uncustomised.
